## 注意事項
- 本品はリポバッテリーを使用しています。リポバッテリーには発火等の危険性があるため十分に注意してご使用ください。
- ロータリーエンコーダーにスイッチ機能はございません。また、滑らかに回るようトルクを外すカスタムをしています。
- ケースは3Dプリンタ製のため積層痕が目立つ場合や、バリが残っている場合がございます、ご了承ください。  
- クリアケースは長期使用により黄変の可能性がございます、ご了承ください。
